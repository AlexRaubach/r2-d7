# R2-D7
A bot for xwingtmg.slack.com.

## Features
- Detects links to lists from <http://geordanr.github.io/xwing/>, <http://xwing-builder.co.uk/build> and <http://x-wing.fabpsb.net> and print them in chat
- Card lookup via [[]] queries
